## 🌟 About This Project

**Web Gerabah 2.0** adalah platform digital yang mempromosikan dan melestarikan kerajinan **gerabah tradisional Indonesia**. Website ini menggabungkan keindahan seni tradisional dengan teknologi modern menggunakan **Anti Gravity** untuk menciptakan pengalaman visual yang memukau dan interaktif.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🚀 **Anti Gravity Animations** | Efek floating, parallax, dan motion yang halus |
| 🏺 **Product Catalog** | Koleksi lengkap gerabah dari berbagai daerah |
| 🎨 **Interactive Gallery** | Galeri dengan efek hover dan gravity-defying transitions |
| 📱 **Responsive Design** | Tampilan optimal di desktop, tablet, dan mobile |
| 🛒 **Shopping Cart** | Sistem keranjang belanja yang mudah digunakan |
| 💬 **Customer Reviews** | Review dan rating dari pelanggan |
| 🔍 **Search & Filter** | Cari produk berdasarkan kategori dan harga |
